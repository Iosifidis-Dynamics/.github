ChatBot-with-Python

Ένα εκπαιδευτικό project ανοιχτού κώδικα που δείχνει πώς να δημιουργήσετε ένα βασικό chatbot χρησιμοποιώντας Python και ένα απλό μοντέλο AI.

Στόχος: Να βοηθήσει τους αρχάριους να κατανοήσουν τα θεμέλια του NLP, τη λογική συνομιλίας και τη δομή ενός chatbot.

Τεχνολογίες: Python, AI, NLP

Guess-Number-Game

Ένα φιλικό προς αρχάριους παιχνίδι "Μάντεψε τον Αριθμό". Ο παίκτης προσπαθεί να μαντέψει έναν τυχαίο αριθμό (1-10) μέσα σε 3 προσπάθειες.

Στόχος: Εξαιρετική άσκηση για την εκμάθηση βασικών εννοιών της Python, όπως loops, conditionals (if/else), user input και random number generation.

Τεχνολογίες: Python
